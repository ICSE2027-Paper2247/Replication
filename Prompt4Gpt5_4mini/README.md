# Prompt Templates

All LLM-based tasks in this study were conducted with GPT-5.4 mini to maintain
model consistency across the analysis pipeline. The outputs of these tasks were
carefully validated through manual inspection, sampling-based evaluation, or
independent annotation, as described in the paper.

This directory contains the prompt templates used for the LLM-based steps in
the study, including multi-licensing identification, multi-licensing target
taxonomy construction, license-term taxonomy construction, multi-licensing
relationship classification, and license-term judgment.
